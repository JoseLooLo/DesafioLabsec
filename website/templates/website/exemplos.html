Exemplos

Serão mostradas alguns exemplos de Requisições para exemplificar o uso da API.
Os exemplos são na linguagem Python e utilizando a library requests.
Para instalar a library requests use pip install requests
*Para problemas com CSRF, consulte o ultimo bloco.

Resumo Criptográfico

import requests
send = {
'arquivo' : (nomeArquivo, open(nomeArquivo, 'rb'), 'multipart/form-data')
}
requests.post('http://localhost:8000/getresumo/', files = send)

import requests
resumos = requests.get('http://localhost:8000/getresumo/')
resumos.json()

Chaves Assimétricas

import requests
